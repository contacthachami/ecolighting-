```
Dimensions: 5cm x 5cm
Resolution: 300 DPI
Format: PNG or Vector (SVG)
Error Correction: High (30%)
Placement: Bottom corner or dedicated section

```
Dimensions: 10cm x 10cm minimum
Resolution: 300 DPI
Format: Vector (SVG preferred)
Error Correction: High (30%)
Placement: Prominent, eye-level
```

#### Product Packaging:

```
Dimensions: 3-4cm x 3-4cm
Resolution: 600 DPI (for small packaging)
Format: Vector (SVG)
Error Correction: High (30%)
Material: Weatherproof if outdoor product
```

```
Dimensions: 15-20cm x 15-20cm
Format: Vector (SVG)
Material: Vinyl, weatherproof
Error Correction: Medium-High (25-30%)
Placement: Eye level, well-lit area

## ğŸ–¨ï¸ Print Specifications

### Paper Types:

**Business Cards:**

- Material: 350gsm cardstock, matte or silk finish
- Coating: Optional UV spot coating on QR only

**Flyers:**

- Material: 150-200gsm glossy or matte
- Size: A5 or A6

**Stickers/Labels:**

- Material: Vinyl (weatherproof)
- Coating: Laminated for durability
- Adhesive: Permanent or removable

**Posters:**

- Material: 200gsm poster paper or vinyl
- Size: A3, A2, or custom
- Coating: Optional lamination

### File Formats to Provide:

**For Print:**

1. **Vector**: SVG or AI (preferred for scalability)
2. **High-Res**: PNG at 600 DPI
3. **PDF**: PDF/X-1a or PDF/X-4 (for professional printing)

**For Digital:**

1. **Web**: PNG at 72-96 DPI (for screens)
2. **Social Media**: 1080x1080px PNG
3. **Email**: PNG, max 500KB file size

## ğŸ“² Testing Protocol

### Before Final Print:

1. **Generate Test QR Code**
   - Use online generator
   - Encode your deployed URL
   - Download high-resolution version

2. **Print Test Copy**
   - Print on regular paper
   - Scale to actual size
   - Test scanning

3. **Scan Testing**
   - Test with iPhone (Camera app)
   - Test with Android (Camera app)
   - Test with WhatsApp scanner
   - Test in different lighting conditions
   - Test from different distances (20cm - 100cm)

4. **Visual Inspection**
   - Check logo clarity
   - Verify color accuracy
   - Ensure text is readable
   - Check alignment

5. **Final Approval**
   - Get team approval
   - Verify with designer
   - Check with print shop

## ğŸ”§ Technical Settings for QR Generation

### Recommended QR Code Generator Settings:

**Error Correction Level:**

- **High (30%)** - Recommended
  - Allows logo in center
  - Damaged code still scans
  - Best for printing

**QR Code Type:**

- **Static URL** - Recommended
  - Direct link to your website
  - No intermediary service
  - Faster loading

**Image Format:**

- **Vector (SVG)** - Best for print
- **PNG (600 DPI)** - Alternative for high-quality print
- **PNG (300 DPI)** - Minimum for print

**Color Settings:**

- **Foreground**: #0a6b3c (or darker)
- **Background**: #ffffff (white)
- **Ensure contrast ratio**: Minimum 4.5:1

## ğŸ’¡ Pro Tips

1. **Always test before mass printing**
2. **Use high error correction** when adding logo
3. **Keep logo max 20%** of QR area
4. **Maintain white border** (quiet zone) of 4-6mm
5. **Use vector format** for scalability
6. **Dark on light** works best (not light on dark)
7. **Avoid gradients** in QR code blocks
8. **Print test on actual material** (vinyl, paper, etc.)
9. **Consider environmental factors** (outdoor = weatherproof)
10. **Update URL, not QR** (keep same QR, change website content)
